# RevOps Diagrams

Interactive swimlane diagrams for the Goflow Core GTM RevOps Buildout project.

## What's here

`index.html` — Interactive account journey diagram showing the paid account flow with a toggleable Core (freemium) overlay. Self-contained HTML file with no external dependencies.

**Toggle off:** Paid account journey across six system lanes (Web, HubSpot, Sales/CS, Goflow, Arrows, QuickBooks). All stages fully measurable.

**Toggle on:** Core freemium flow overlaid on the dimmed paid base. Color-coded by measurement status — red for blind spots, amber for partial coverage, green for working infrastructure. Schema pills show HubSpot property names, object types, and gap indicators.
